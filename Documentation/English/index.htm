Goal

Vczh Library++ 3.0 for C++ contains script engines, libraries and tools for building C++ programs who needs to process data and string in an environment without extreme requirements of performance, or needs a fast script engine for doing complex tasks.
In this project we are planning to have

  • A basic library for C++. It has some basic constructions, collections, a stream system, a regular expression library, a flexible syntax analyzer and a easy-to-setup reflection system for C++. Other C++ projects is built on the basic library.
  • A script engine. Coded in C++, but it is designed to have both native and managed interfaces.
  • An IDE for debugging your script. It is designed to be extensible. The IDE project results in some .NET assemblies. You can customize it, build a separate program or integrate it into your project.
