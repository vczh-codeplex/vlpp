Func<R(T1, T2, ...)>

Func用来抽象普通函数、类成员函数和其他可执行对象的函数对象。由于C++的限制，Func只支持0到10个参数。

int MyAdd(int a, int b)
{
    return a+b;
}

int main()
{
    Func<int(int,int)> add=MyAdd;
    Console::WriteLine(itow(add(1,2)));
    return 0;
}

不同参数个数的Func对象其成员也有一点变化。不失一般性，下面的描述都以Func<R(T0, T1, T2)>为标准。
