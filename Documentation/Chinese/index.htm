目标

Vczh Library++是一个可以同时支持各种等级的各种语言的通用脚本平台，并提供一系列的库让一门新的语言可以很容易地一直到这个平台上。不仅如此，该平台还包含一个强大的数据处理和字符串处理库。

工程结构

本工程由以下解决方案组成

<Documentation>
    <Chinese> 中文文档资料。
    <English> 英文文档资料。
    Documentation.sln 为了在Visual Studio中更方便地管理文档文件而建立的解决方案。
<Library> 目标库代码。
<Tools>
    <AutoShell>
        AutoShell.sln 脚本引擎的集成开发环境，未完成。
    <FpMacro>
        FpMacro.sln Functional Macro被用来生成C语言的宏不能或很难生成的代码。该工程的Release版本必须被放在<Tools>\FpMacro.exe下。
<UnitTest>
    UnitTest.sln 单元测试工程。

本工程由以下几部分组成

  1. Vczh Library++ 核心库

  2. Functional Macro System

  3. Auto Shell

  4. Script
