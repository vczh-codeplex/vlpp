目标

Vczh Library++包含一系列的库和工具，用来更加容易地开发用于在对性能要求没有那么极端的数据处理程序，并且提供了一个快速的脚本引擎来辅助完成复杂的任务。
我们在工程中的计划包括：

  • C++基础类库，包含基础设施、容器、流系统、正则表达式引擎、可配置语法分析器和一个易于建立和使用的C++反射系统（非全自动）。
  • 一个拥有本地和托管两种接口的脚本引擎。
  • 一个用于调试脚本的集成开发环境。这个集成开发环境使用.NET编写，可以被修改成独立程序，或者集成到其他工程里面去。

    • 工程结构

    本工程由以下解决方案组成

    <Documentation>
        <Chinese> 中文文档资料。
        <English> 英文文档资料。
        Documentation.sln 为了在Visual Studio中更方便地管理文档文件而建立的解决方案。
    <Library>
        <Tools>
        <AutoShell>
            AutoShell.sln 脚本引擎的集成开发环境，未完成。
        <FpMacro>
            FpMacro.sln Functional Macro被用来生成C语言的宏不能或很难生成的代码。该工程的Release版本必须被放在<Tools>\FpMacro.exe下。
        <OfficeIcon>
            OfficeIcon.sln 开发用的工具：浏览某指定文件夹下的图标文件。
        <UnitTest>
            UnitTest.sln 单元测试工程。

    本工程由以下几部分组成

    1. Vczh Library++ 核心库

    2. Functional Macro System
